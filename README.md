# ArchonCrates
An advanced Crates plugin for PocketMine-MP.

## How to use ArchonCrates

/key <Player> to give a key.
/keyall <type> to give a key to all the players on the server
/keylist for a list of keys

### How to spawn a crate
Go to the config.yml, put the world where you want the crates,

put the coordinates and restart your server.

### ArchonCrates is fully compatible and usable with the following plugins: PureChat, AdvancedKits, ArchonBanknotes.
